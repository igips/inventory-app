# Inventory App

Inventory app made with Express and MongoDB

[Live Demo](https://cryptic-bastion-99496.herokuapp.com/) :point_left:

## Features
* Responsive design
* Add/Edit/Delete items and categories

## Built with
- [Express](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)
- [EJS](https://ejs.co/)
